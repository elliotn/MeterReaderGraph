Meter Reader v1.0.3

Purpose:
Graph water usage. Get notified when too much water has been used on the previous day. View statistics, such as last 30 days usage or specify start and ends to compare usage against bill.

Note:
This app itself does not communicate with smart meters. It simply graphs the data and can send notifications.

Setup:
In order for this app to show usage, you will need the following:
  • hardware (an rtl-sdr compatible device, such as this one.)

  • software for reading the meter (use JSON output), and a web server to host the daily log.

  • consumption log (here you can find a script to generate/append to daily log and a cronjob to run the script daily.)

Usage:
After a few readings have occurred, set the URL in the app, and graph the data.

You can also tell the app to automatically notify you if some threshold is reached via settings. (E.g., too much water was consumed.)
